Integraciones

YouTube Data API v3 - Sistema preparado

La plataforma esta preparada para conectarse a la YouTube Data API v3 de Google. Actualmente funciona con listas de videos estaticas (sin consumir cuota), pero el sistema esta disenado para que cualquier usuario pueda activar la API real y cargar playlists dinamicas.

Que ofrece la YouTube Data API v3?

  • Acceso programatico a videos, playlists, canales y estadisticas de YouTube.
  • Busqueda de contenido por palabras clave, canal o ID de lista.
  • Obtencion de miniaturas, titulos, descripciones y metadatos en tiempo real.
  • Compatible con JavaScript mediante solicitudes REST (fetch / axios).

Como obtener tu API Key

  1. Ve a Google Cloud Console.
  2. Crea un proyecto nuevo o selecciona uno existente.
  3. Activa la API YouTube Data API v3 en la biblioteca de APIs.
  4. Ve a Credenciales y genera una Clave de API (restringe por HTTP referer para seguridad).
  5. Copia tu clave y configurala en el archivo de variables del proyecto.

Cuotas y limites

Google asigna 10.000 unidades de cuota al dia por defecto (gratuito). Cada operacion consume unidades: una busqueda cuesta 100 unidades, obtener detalles de una lista 1 unidad por elemento, y la lista de videos de un canal 1-3 unidades. Con uso moderado, 10.000 unidades cubren miles de visitas diarias.

Estado de la plataforma

  • Fase actual: listas estaticas definidas en src/js/data/youtube-playlist.js. Sin necesidad de API key ni cuota.
  • Preparacion: el modulo src/js/services/youtube-service.js ya esta estructurado para aceptar peticiones a https://www.googleapis.com/youtube/v3/.
  • Proximo paso: sustituir las listas estaticas por llamadas a la API usando la key del usuario, o crear un archivo .env / config.js donde cada usuario introduzca su propia clave.
